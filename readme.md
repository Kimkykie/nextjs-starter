## Quick Overview

```sh
$ git clone https://github.com/Kimkykie/nextjs-starter.git
$ cd nextjs-starter
$ npm or yarn install
For dev run
$ npm run dev
Open your browser on localhost:7777/
```

## Note
You can change your preferred port on the package.json scripts

```sh
"scripts": {
    "dev": "next -p [PREFERRED_PORT]",
    ...
},
```
